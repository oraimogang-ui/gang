# Vertex Studio Marketing Site

A fully responsive marketing website for the fictional digital agency Vertex Studio. It showcases services, selected work, testimonials, pricing, and a contact form with client-side validation.

## Getting started

Open `index.html` in your browser to explore the site locally. All assets are self-contained, and no build step is required.

## Features

- Sticky navigation with responsive mobile menu
- Hero, services, portfolio, metrics, testimonials, pricing, and contact sections
- Auto-playing testimonial slider with pause on hover
- Accessible contact form with real-time validation feedback
- Modern aesthetic with responsive layout and typography
